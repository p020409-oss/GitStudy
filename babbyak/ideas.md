## 아이디어 2: "스타트업 IR 덱 패러디" ← 선택
<response>
<text>
**Design Movement**: Apple/Notion 스타일 + 스타트업 랜딩페이지 + MZ 밈
**Core Principles**:
- 진짜 스타트업 랜딩페이지처럼 세련됐는데 내용이 킹받음
- 다크모드 기반, 민트/청록 포인트 컬러
- 카드 UI + 큰 라운드 버튼
- 스크롤 fade-in 애니메이션으로 진지한 척
**Color Philosophy**: 
- 배경: #0a0a0f (거의 검정)
- 카드: #111118 (약간 밝은 다크)
- 포인트: #00d4aa (민트/청록) - 세련되고 MZ스러운 컬러
- 텍스트: 흰색 계열
- 서브텍스트: 회색 계열
**Layout Paradigm**: 
- 풀스크린 섹션들이 세로로 쌓이는 구조
- 중앙 정렬이지만 카드들은 그리드로 배치
- 모바일 우선 설계
**Signature Elements**:
- 민트색 그라디언트 글로우 효과
- 카드에 미세한 테두리 (border: 1px solid rgba(255,255,255,0.08))
- 배지/태그 스타일 강조 텍스트
**Interaction Philosophy**:
- 카드 hover 시 살짝 위로 뜨는 효과 + 민트 글로우
- 버튼 hover 시 민트 배경으로 전환
- 토스트 메시지로 웃긴 반응
**Animation**:
- Framer Motion으로 스크롤 시 fade-in + slide-up
- 스태거 애니메이션으로 카드들 순차 등장
- 버튼 active 시 scale(0.97) 피드백
**Typography System**:
- Pretendard 폰트 (Google Fonts CDN)
- 제목: 700-800 weight, 큰 사이즈
- 본문: 400-500 weight
- 강조: 민트 컬러 + 600 weight
</text>
<probability>0.08</probability>
</response>
